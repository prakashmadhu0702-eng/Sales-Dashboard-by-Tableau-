# 📊 Sales Performance Dashboard - Tableau

## Project Overview
This project presents an interactive **Sales Performance Dashboard** built using Tableau.  
The dashboard provides insights into sales trends, profit distribution, and product performance to help understand overall business performance.

The goal of this project is to analyze sales data and present meaningful insights through clear and interactive visualizations.

## Objectives
- Analyze overall sales and profit performance
- Identify top-performing products and categories
- Understand sales distribution across different regions
- Track monthly sales trends
- Provide insights for better business decision-making

## Tools Used
- Tableau – Data visualization and dashboard creation
- Microsoft Excel / CSV – Data source and data preparation

## Dashboard Features
- Total Sales, Profit, and Quantity KPIs
- Sales by Category and Sub-Category
- Region-wise Sales Performance
- Monthly Sales Trend Analysis
- Top Performing Products
- Interactive Filters for deeper analysis

## Key Insights
- Identified categories generating the highest revenue
- Analyzed regional sales contribution
- Discovered seasonal patterns in sales trends
- Compared profit and sales performance across products

## Dataset
The dataset contains sales transaction information including:
- Order Date
- Region
- Category
- Sub-Category
- Sales
- Profit
- Quantity
